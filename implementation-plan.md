# 实施计划 — API 中转站

## 阶段 1：环境搭建（1-2 天）

### Step 1: 服务器准备
- [ ] 购买云服务器（推荐：腾讯云/阿里云轻量，香港节点）
- [ ] 配置 SSH 密钥登录
- [ ] 安装 Docker + Docker Compose
- [ ] 配置防火墙（只开放 80/443/SSH）

### Step 2: 部署 New API
- [ ] 修改 `docker-compose.yml`（改密码、改时区）
- [ ] `docker-compose up -d` 启动
- [ ] 访问 `http://服务器IP:3000` 验证
- [ ] 修改管理员默认密码

### Step 3: 域名与 SSL
- [ ] 购买域名
- [ ] DNS 解析到服务器 IP
- [ ] 配置 Nginx/Caddy 反向代理
- [ ] 申请 SSL 证书（Let's Encrypt 或 Cloudflare）

## 阶段 2：渠道配置（1-2 天）

### Step 4: 对接上游供应商
按优先级逐个添加渠道：

**第一批（必须有）**：
- [ ] OpenAI（GPT-4o, GPT-4o-mini）
- [ ] DeepSeek（V3, R1）— 国产，便宜
- [ ] 通义千问（Qwen-Max, Qwen-Plus）

**第二批（尽快添加）**：
- [ ] Anthropic Claude（Sonnet, Haiku）
- [ ] Google Gemini（2.0 Flash, Pro）

**第三批（按需）**：
- [ ] 月之暗面 Kimi
- [ ] 智谱 GLM
- [ ] 硅基流动（SiliconFlow 多模型）

### Step 5: 模型定价
- [ ] 配置每个模型的倍率（售价 = 成本 × 倍率）
- [ ] 设置用户分组（免费/普通/VIP）
- [ ] 配置各分组的价格折扣

### Step 6: 渠道健康配置
- [ ] 每个核心模型至少 2 个渠道（主 + 备）
- [ ] 配置自动测试间隔
- [ ] 配置 failover 策略

## 阶段 3：支付与用户（2-3 天）

### Step 7: 支付接入
- [ ] 注册易支付/Stripe
- [ ] 配置支付回调
- [ ] 测试充值流程
- [ ] 设置充值档位（¥10/30/50/100/500）

### Step 8: 用户系统
- [ ] 配置注册方式（邮箱/GitHub/Discord）
- [ ] 设置新用户赠送额度（$0.5）
- [ ] 编写用户使用文档
- [ ] 配置内容审核规则

### Step 9: API 文档
- [ ] 编写快速开始指南
- [ ] 提供各语言示例代码（Python/curl/Node.js）
- [ ] 说明兼容 OpenAI SDK 的用法
- [ ] FAQ 常见问题

## 阶段 4：运营上线（1-2 天）

### Step 10: 监控告警
- [ ] 配置渠道可用性监控
- [ ] 设置余额不足告警
- [ ] 配置异常用量告警
- [ ] 日志收集与分析

### Step 11: 推广启动
- [ ] V2EX 发帖
- [ ] 掘金写教程文章
- [ ] GitHub 项目推荐
- [ ] 建立 Telegram/Discord 群

### Step 12: 持续优化
- [ ] 每周检查各渠道成本/利润
- [ ] 根据用户需求添加新模型
- [ ] 优化热门模型的响应速度
- [ ] 调整定价策略

## 里程碑

| 里程碑 | 预计时间 | 标志 |
|---|---|---|
| 🏗️ 服务上线 | 第 1 周 | 域名可访问，API 可用 |
| 👤 第一个付费用户 | 第 2 周 | 有人充值 |
| 📈 日活 10+ | 第 1 月 | 稳定运行 |
| 💰 月收入 ¥1000+ | 第 2 月 | 覆盖成本 |
| 🚀 月收入 ¥5000+ | 第 3-6 月 | 开始盈利 |

## 总工期估算
- **最快上线**：3-5 天（如果服务器/域名/支付已准备好）
- **稳妥上线**：1-2 周（含测试和文档）
